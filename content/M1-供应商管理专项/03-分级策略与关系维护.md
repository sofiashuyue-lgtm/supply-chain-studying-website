# 03 分级策略与关系维护

> **📌 知识层级说明**
> 🟢 **基础必掌握** — 入职前、面试前必须理解
> 🟡 **进阶了解** — 有基础后扩展，理解逻辑即可

---

## 一、为什么要对供应商分级？🟢 基础必掌握

一家中型制造企业可能有 200～2000 家供应商，不可能对所有供应商投入同样的时间和资源。

**分级管理的核心逻辑**：把有限的采购资源（时间、精力、关系维护成本）集中在最重要的供应商上，对低重要性供应商则简化管理。

---

## 二、Kraljic 矩阵：供应商分级的黄金工具 🟢 基础必掌握

**Kraljic 矩阵（Kraljic Matrix）**：由麦肯锡顾问 Peter Kraljic 于1983年在《哈佛商业评论》提出，是全球采购领域最权威的供应商分类框架，至今被全球企业广泛使用。

### 矩阵结构

两个维度：
- **X 轴：采购金额（Spend）**：该类物料在总采购支出中的占比
- **Y 轴：供应风险（Supply Risk）**：该物料的供应难度、替代难度、单一来源风险

```
        高
        │  【瓶颈型】          【战略型】
供      │  Bottleneck           Strategic
应      │  高风险·低金额         高风险·高金额
风      │  ─────────────────────────────────
险      │  【常规型】          【杠杆型】
        │  Routine              Leverage
        │  低风险·低金额         低风险·高金额
        低────────────────────────────────→
                   低           高
                        采购金额
```

### 四类供应商详解

#### 🔴 战略型（Strategic）— 高风险 · 高金额
**特征**：采购金额大，替代困难，对生产至关重要
**典型物料**：核心芯片（CPU/GPU）、定制模具、关键原材料
**管理策略**：
- 建立长期战略合作协议（3～5年）
- 高层定期互访，建立深度伙伴关系
- 联合研发、共同开发下一代产品
- 必须保留备份方案（Second Source 或内部自制）

#### 🟡 杠杆型（Leverage）— 低风险 · 高金额
**特征**：采购金额大，但市场供应充足，替代容易
**典型物料**：标准电阻/电容、通用螺丝、大宗原材料
**管理策略**：
- 充分利用竞争优势，多家对比询价
- 批量集采降低价格，争取最优条件
- 无需深度关系维护，主要靠价格竞争驱动

#### 🟠 瓶颈型（Bottleneck）— 高风险 · 低金额
**特征**：采购金额不大，但供应商稀缺或替代困难
**典型物料**：特种认证材料、专利工艺配件、小批量定制件
**管理策略**：
- 提前备货，建立安全库存缓冲
- 与供应商维持稳定关系，不要因小利得罪
- 积极寻找替代方案（替代料认证）

#### ⚪ 常规型（Routine）— 低风险 · 低金额
**特征**：金额小，替代容易，管理成本高于价值
**典型物料**：办公耗材、标准五金件、通用包装材料
**管理策略**：
- 简化采购流程（电商平台、框架协议）
- 推行 P-Card（采购卡）或自助采购降低管理成本
- 不需要深度管理，按需采购即可

## 三、单一来源（Single Source）风险管理 🟢 基础必掌握

### 什么是 Single Source？

**Single Source（单一来源）**：某个物料只有一家合格供应商可以提供。

> 注意区分：
> - **Sole Source（唯一来源）**：全市场只有一家能做，无法改变（如专利技术）
> - **Single Source（单一来源）**：市场上有多家，但企业只选了一家（可以改变）

### Single Source 的双面性

| 维度 | 优势 | 风险 |
|------|------|------|
| **采购管理** | 关系深度好，沟通成本低 | 供应商"绑架"采购方，失去议价能力 |
| **供应安全** | 供应商对你的业务优先级高 | 一旦出问题（火灾、停产、倒闭），立即断供 |
| **成本** | 量集中，单价可能更低 | 缺乏竞争，长期价格可能偏高 |

### 主动管理 Single Source 的方法

```
识别所有 Single Source 物料
        ↓
按风险程度排序（供应金额 × 替代难度）
        ↓
Top 风险物料 → 启动 Second Source 开发
中等风险 → 增加安全库存作为缓冲
低风险 → 维持现状，定期监控
```

## 四、Second Source（备份供应商）开发 🟡 进阶了解

**Second Source**：针对 Single Source 物料，开发第二家合格供应商作为备份，以降低断供风险。

### 开发节奏

| 阶段 | 行动 | 目的 |
|------|------|------|
| **认证阶段** | 启动备份供应商准入评估 | 让备份供应商进入 AVL |
| **小量试单** | 每6个月给备份供应商下一定比例订单 | 保持关系活跃，验证量产能力 |
| **常态双源** | 主供应商 70%，备份 30% | 保证供应安全，同时维持竞争 |

### 份额分配参考原则

| 供应商角色 | 份额比例 | 说明 |
|---------|---------|------|
| 主供应商（Primary） | 60～80% | 综合评分最优，优先保证份额 |
| 备份供应商（Secondary） | 20～40% | 维持活跃，防止主供断货 |

> 💡 **面试加分点**：能说出"我会根据 Scorecard 绩效动态调整主备份额比例——主供表现好则加仓，表现差则向备份转移份额"，体现量化管理思维。

## 五、不同等级供应商的关系维护节奏 🟡 进阶了解

| 供应商等级 | 拜访频率 | 沟通方式 | 合同类型 |
|---------|---------|---------|---------|
| **战略型** | 每月例会 + 季度 QBR + 年度联合规划 | 高层互访 + 工作层紧密对接 | 3～5年长协 + 保密协议 |
| **杠杆型** | 季度 QBR | 工作层为主 | 1年框架合同 |
| **瓶颈型** | 半年一次 | 定期沟通，关系维护 | 年度合同 + 备货协议 |
| **常规型** | 按需 | 线上为主 | 框架协议或单次 PO |

## 六、中国制造业的供应商关系特点 🟡 进阶了解

| 特点 | 说明 |
|------|------|
| **关系文化重要性** | 在中小企业中，"人情"仍是维系供应商关系的重要纽带 |
| **大客户主导** | 苹果、华为等大厂在供应商关系中强势主导，供应商话语权弱 |
| **国产替代浪潮** | 近年来半导体等领域"去美化"政策推动，大量外资供应商被替换为国内供应商，是供应商结构重大变化 |
| **产业集群** | 中国供应商高度集中（深圳电子、宁波汽配、东莞模具），地理集聚提高了协同效率 |
