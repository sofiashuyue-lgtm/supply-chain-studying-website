# 01 供应商准入与认证

> **📌 知识层级说明**
> 🟢 **基础必掌握** — 入职前、面试前必须理解
> 🟡 **进阶了解** — 有基础后扩展，理解逻辑即可
> 🔴 **暂缓** — 工作后遇到实际场景再深入

## 一、什么是供应商准入？为什么重要？🟢 基础必掌握

**供应商准入（Supplier Qualification / Onboarding）**：企业在与新供应商正式合作之前，对其资质、能力、风险进行系统性评估和认证的过程。

### 为什么必须做准入？

不做准入直接采购，就像不体检就雇人。风险包括：
- 质量失控（来料不合格导致停线或产品召回）
- 交付风险（供应商产能不足，无法按时交货）
- 合规风险（供应商存在违规行为，连带品牌方受罚）
- 财务风险（供应商经营不稳定，货款预付后跑路）

> 💡 **面试常见问题**："你们公司是如何开发一个新供应商的？" ——这就是在考查你对准入流程的理解。

---

## 二、AVL（合格供应商名录）🟢 基础必掌握

**AVL（Approved Vendor List，合格供应商名录）**：企业通过准入评估的供应商名单，只有进入 AVL 的供应商才有资格接受采购订单。

### AVL 的核心作用

```
新供应商申请
     ↓
  准入评估
     ↓
通过 → 进入 AVL → 可以下 PO 采购
失败 → 不进 AVL → 无法合作（或进入整改期）
```

### AVL 的分类管理

| 状态                    | 含义                  |
| --------------------- | ------------------- |
| **合格（Active）**        | 通过全部评估，可正常采购        |
| **条件合格（Conditional）** | 部分指标待改善，限量或试用采购     |
| **整改中（Probation）**    | 发现重大问题，暂停采购，等待整改完成  |
| **淘汰（Disqualified）**  | 严重违规或长期不达标，从 AVL 移除 |

### 进入 AVL 的难度因素

- **行业越严格，门槛越高**：汽车（IATF 16949）> 半导体 > 消费电子 > 一般制造
- **大客户壁垒**：进入苹果、华为、宁德时代的 AVL 是中小供应商的重要背书
- **一旦进入，有维护成本**：定期重审，绩效不达标会被移除

## 三、供应商准入评估流程 🟢 基础必掌握

### 标准流程（四阶段）

```
第一阶段：信息收集（桌面评审）
         ↓
第二阶段：文件审核（资质核验）
         ↓
第三阶段：现场审核（工厂审厂）
         ↓
第四阶段：样品/试单验证 → 准入决策
```

---

### 第一阶段：信息收集

通过 **RFI（信息征询书）** 或供应商自评表，收集以下基础信息：

- 营业执照、注册资本、经营范围
- 工厂地址、员工规模、技术人员占比
- 主要客户清单（参考质量口碑）
- 年营业额（判断财务稳定性）

---

### 第二阶段：文件审核（Document Review）

核查以下关键文件：

| 文件类别       | 具体内容                                       |
| ---------- | ------------------------------------------ |
| **质量体系认证** | ISO 9001、IATF 16949（汽车）、ISO 13485（医疗，暂不展开） |
| **环保合规**   | RoHS / REACH 符合性声明、危险物质管理报告                |
| **财务文件**   | 近3年审计报告或纳税证明（判断财务健康度）                      |
| **知识产权**   | 无知识产权纠纷声明                                  |
| **保险**     | 产品责任险、工伤险是否齐全                              |

---

### 第三阶段：现场审厂（Factory Audit / On-site Audit）🟡 进阶了解

**审厂**是准入流程中含金量最高的一步，需要采购、质量、工程多部门联合参与。

#### 审厂核查重点

| 维度        | 具体检查项                        |
| --------- | ---------------------------- |
| **管理层承诺** | 质量方针是否明确、管理层是否重视品质           |
| **生产设备**  | 关键设备是否满足工艺要求，保养记录是否完整        |
| **过程控制**  | SOP（标准作业程序）是否存在并被执行          |
| **检验能力**  | IQC/IPQC/OQC 体系是否健全，测量设备是否校验 |
| **仓储管理**  | 物料先进先出（FIFO）、防静电、温湿度控制       |
| **供应商管理** | 供应商是否对其上游也做了管控               |

#### 审厂评分与结论

| 得分 | 结论 |
|------|------|
| 90分以上 | 一类供应商，直接准入 |
| 70～89分 | 二类供应商，条件准入，限期整改 |
| 60～69分 | 整改后复审 |
| 60分以下 | 拒绝准入 |

---

### 第四阶段：样品与试单验证

- **样品（Sample）**：要求供应商提供样品，按技术规格做全项检验
- **PPAP（生产件批准程序）**：汽车行业要求供应商提交量产能力证明（🔴 暂缓，汽车行业深入再学）
- **试单（Pilot Order）**：小批量试采购，验证量产质量和交期

---

## 四、AVL 的日常维护 🟡 进阶了解

进入 AVL 不是一劳永逸，需要定期重审（通常每年1次）：

- 绩效评分持续低于阈值 → 触发整改或降级
- 发生重大质量/交付事故 → 即时降级或暂停
- 认证证书过期未续 → 暂停采购资格
- 供应商发生重大经营变化（被收购、关键人员离职）→ 重新评估

---

## 五、中国市场特点 🟡 进阶了解

| 特点          | 说明                                           |
| ----------- | -------------------------------------------- |
| **外资大厂门槛高** | 进入华为、比亚迪、大疆的 AVL 需要通过严格的 SQE 审厂，通常需要6个月以上    |
| **认证优先**    | IATF 16949（汽车）、ISO 9001（通用制造）是进入大厂 AVL 的基础门票 |
| **关系文化影响**  | 部分中小企业准入流程不规范，存在"走后门"现象；大厂准入流程相对规范透明         |
| **本地化优先**   | 近年来政策和供应链安全导向推动"国产替代"，本土供应商准入机会增加            |
