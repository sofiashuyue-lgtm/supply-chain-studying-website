## 一、什么是 TCO？为什么比"报价"更重要？🟢 基础必掌握

**TCO（Total Cost of Ownership，总拥有成本）**：评估一个采购决策的**全生命周期成本**，而不仅仅是采购单价。

### 单价思维 vs TCO 思维的对比

**场景**：选择电子元件供应商

| 评估维度 | 供应商A | 供应商B |
|---------|---------|---------|
| 单价 | **¥10.00（更低）**| ¥11.00 |
| PPM 不良率 | 2,000 PPM | 200 PPM |
| OTIF 交货率 | 88% | 98% |
| Lead Time | 14 天 | 7 天 |
| 距离 | 1,500 km | 300 km |

**如果只看单价 → 选 A**  
**如果看 TCO → 可能选 B**：

```
供应商A 的隐性成本：
  + 不良品返工/退货成本（PPM 2000 → 每百万颗报废2000颗）
  + 频繁交期延误导致的停线损失
  + 长 Lead Time 需要更多安全库存占用资金
  + 远距离运输费用更高
  ──────────────────────
  实际 TCO 可能超过 ¥13.00
```

> 💡 **面试高频问题**："你如何评估供应商的价格竞争力？"  
> ✅ 正确答案：不只看单价，要看 TCO（总拥有成本），包括质量、交付、运输、库存等全链条成本。

## 二、TCO 的完整构成 🟢 基础必掌握

TCO 通常分为三大类成本：

### 2.1 采购前成本（Pre-Transaction Costs）

```
供应商开发成本：
  + 寻源费用（差旅、RFQ 处理时间）
  + 供应商审核费用（审厂、资质审查）
  + 样品认证费用（打样、测试）
  + 合同谈判成本（法务、时间）
```

### 2.2 交易成本（Transaction Costs）— 核心

```
直接采购成本：
  + 采购单价 × 数量           ← 最容易被关注
  + 包装费用
  + 关税 / 进口税
  + 国际 / 国内运费
  + 保险费
  + 报关代理费

间接采购成本：
  + 订单处理成本（下单、跟单人工成本）
  + 到货检验（IQC）成本
  + 不合格品处理成本（返工、退货、索赔）
  + 交期延误导致的停线损失
  + 紧急采购溢价（急单时价格通常高 10%～30%）
```

### 2.3 采购后成本（Post-Transaction Costs）

```
使用阶段成本：
  + 库存持有成本（安全库存 × 持库率）
  + 呆滞库存损失
  + 质量保证期内的保修成本
  + 供应商管理成本（QBR、绩效回顾）

退出成本：
  + 供应商切换成本（重新认证新供应商）
  + 设计变更成本（物料更换引起的 BOM 变更）
```

## 三、TCO 计算模板 🟡 进阶了解

### 标准化 TCO 计算表

| 成本项 | 供应商A | 供应商B | 备注 |
|--------|---------|---------|------|
| **采购单价** | 10.00 | 11.00 | 元/个 |
| **运费** | 0.80 | 0.30 | 远近不同 |
| **关税** | 0.50 | 0.50 | 相同 |
| **进口代理费** | 0.10 | 0.05 | — |
| **IQC 检验成本** | 0.15 | 0.05 | PPM越高检验越严 |
| **不良品返工成本** | 0.60 | 0.06 | PPM 2000 vs 200 |
| **停线损失分摊** | 0.80 | 0.10 | OTIF 88% vs 98% |
| **安全库存资金成本** | 0.40 | 0.15 | Lead Time 14d vs 7d |
| **供应商管理成本** | 0.20 | 0.08 | 频繁问题处理 |
| **TCO 合计** | **13.55** | **12.29** | — |
| **TCO 差异** | +1.26（贵 10%）| — | 单价看起来便宜，实际更贵 |

## 四、TCO 在战略决策中的应用 🟡 进阶了解

### 4.1 供应商选择决策

不能只比价，要比 TCO。具体数据采集方式：
- 不良成本：历史 PPM 数据 → 换算返工/报废单价
- 停线成本：与生产部门协商确定"停线1小时损失多少"
- 运输成本：实际物流报价

### 4.2 本地化 vs 离岸采购决策

**场景**：某零件
- 国内供应商报价：¥50/个，Lead Time 7 天
- 越南供应商报价：¥38/个，Lead Time 35 天

**表面看差价 24%，很划算**。但 TCO 分析需加上：
- 海运费 + 进口关税
- 更长 Lead Time 需要额外安全库存
- 汇率风险
- 质量风险（新供应商认证成本）
- 供应链韧性风险（疫情/地缘政治）

**结论**：TCO 差距可能缩小到 5%～10%，综合考虑后不一定值得离岸。

### 4.3 自制 vs 外购决策（Make vs Buy）

| 成本项 | 自制 | 外购 |
|--------|------|------|
| 直接材料 | + | — |
| 直接人工 | + | — |
| 制造费用 | + | — |
| 质量控制 | + | 含在采购价中 |
| 设备折旧 | + | — |
| 管理成本 | + | 含供应商管理 |
| 采购价格 | — | + |
| 供应弹性 | 固定产能 | 可扩充 |

TCO 分析帮助决定哪些业务外包更合算。

## 五、关键术语速查 🟢

| 术语 | 解释 |
|------|------|
| **TCO** | Total Cost of Ownership，总拥有成本 |
| **Landed Cost** | 落地成本，货物到达目的地的全部成本（含运、税、保）|
| **Hidden Cost** | 隐性成本，不体现在采购价格中的间接成本 |
| **Make vs Buy** | 自制与外购决策 |
| **Switching Cost** | 切换成本，更换供应商所需的重新认证、设计变更等成本 |
| **Emergency Premium** | 紧急溢价，急单时因交期紧张需额外支付的价格 |
