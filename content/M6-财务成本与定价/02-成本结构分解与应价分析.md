> 🟡 **进阶了解** — 有基础后扩展，理解逻辑即可

## 一、为什么要拆解供应商成本？🟢 基础必掌握

**不了解成本结构，谈判就是瞎砍价。**

当你拆解了供应商的成本构成，你就能：
- 识别哪部分利润空间可压缩（毛利率过高）
- 判断涨价请求是否合理（原材料真的涨了多少？）
- 提出针对性的降本方案（改工艺、换材料、规模化）
- 在谈判中用数据说话，而非凭感觉

> 💡 **转行迁移**：工程造价的**工程成本分解**（人工费 + 材料费 + 机械费 + 管理费 + 利润 + 税金）与供应商成本分解逻辑完全一致。你已经有扎实的基础。

## 二、制造业供应商成本构成 🟢 基础必掌握

### 标准成本分解模型（制造业通用）

```
供应商报价
├── 直接成本（Variable Costs）
│   ├── 直接材料（Direct Material，DM）
│   │   ├── 主材（关键原材料）
│   │   ├── 辅材（辅助材料、耗材）
│   │   └── 包装材料
│   ├── 直接人工（Direct Labor，DL）
│   │   ├── 生产线工人工资
│   │   └── 社保 + 福利
│   └── 外协加工费（如有）
│
├── 制造费用（Manufacturing Overhead，MOH）
│   ├── 设备折旧
│   ├── 水电费
│   ├── 模具摊销
│   └── 质检成本
│
├── 期间费用（Operating Expenses，OPEX）
│   ├── 销售费用（含销售人员薪资）
│   ├── 管理费用（总部分摊）
│   └── 研发费用分摊（如有）
│
├── 财务费用（Financial Costs）
│   └── 资金利息（应收账款融资成本）
│
└── 利润（Profit Margin）
    └── 目标毛利率（通常 8%～25%，因行业而异）
```

### 各成本项占比参考（消费电子精密制造）

| 成本项 | 占比参考 |
|--------|---------|
| 直接材料（DM）| 45%～65% |
| 直接人工（DL）| 8%～15% |
| 制造费用（MOH）| 10%～20% |
| 期间费用（OPEX）| 5%～10% |
| 利润（Margin）| 8%～15% |

## 三、成本模型工具：SHOULD COST 🟡 进阶了解

**Should Cost（应该成本/标准成本）**：采购方基于对成本结构的了解，自行计算出"这个产品的合理成本应该是多少"，作为谈判基准。

### Should Cost 建立步骤

**Step 1：分解物料 BOM，逐项估算原材料成本**

```
产品：某塑料零件（单个）

原材料：
  ABS 塑料：0.05 kg × 市场价 ¥18/kg = ¥0.90
  色母粒：  0.001 kg × ¥25/kg     = ¥0.03
  材料小计：                         ¥0.93
```

**Step 2：估算加工工时和人工成本**

```
注塑成型工时：0.5 分钟/个
  生产线人工成本：¥25/小时 → ¥0.21/个

后处理（去毛刺）工时：0.2 分钟/个
  人工成本：¥0.08/个

人工小计：                           ¥0.29
```

**Step 3：制造费用分摊**

```
设备折旧 + 水电：按人工成本的 80% 估算 → ¥0.23
```

**Step 4：加上期间费用和利润**

```
期间费用：按直接成本的 8% → ¥0.12
利润：按总成本的 10% → ¥0.16

Should Cost = 0.93 + 0.29 + 0.23 + 0.12 + 0.16 = ¥1.73
```

**若供应商报价 ¥2.20**：
→ 溢价约 27%，谈判空间相当大
→ 可以目标 ¥1.85～¥1.95 作为谈判锚点

## 四、应价分析（Price Analysis）🟢 基础必掌握

**应价分析**是评估供应商报价合理性的系统性方法，不依赖完整的成本拆解，而是通过多维度对比判断价格是否合理。

### 常用应价分析方法

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| **竞争性报价比较** | 同时向 3 家以上供应商发 RFQ，对比报价差异 | 标准件、通用件采购 |
| **历史价格趋势分析** | 对比同一供应商历次报价变化趋势 | 判断涨价合理性 |
| **市场公开价格对标** | 参考商品交易所价格（铜、铝）或行业报价指数 | 大宗原材料采购 |
| **Should Cost 模型** | 自建成本模型，推算合理价格区间 | 定制件、非标件 |
| **同类产品价格类比** | 以相似产品的市场价格为参考基准 | 新产品初期报价 |
| **价格指数追踪** | 使用 CPI、PPI、大宗商品指数判断成本变化 | 评估涨价理由 |

## 五、原材料联动：涨价请求的审核 🟡 进阶了解

供应商常以"原材料涨价"为由申请调价。采购人需要核实这一理由是否成立。

### 审核涨价请求的步骤

**Step 1：要求供应商提供成本分解**

供应商必须书面提供：
- 涨价原材料的名称和用量（每单位产品消耗多少）
- 原材料成本在产品总成本中的占比
- 原材料价格变动的证明（发票、市场报价单）

**Step 2：用公式验算涨价幅度的合理性**

$$\text{合理涨价幅度} = \text{原材料成本占比} \times \text{原材料涨价幅度}$$

**示例**：
- 产品总成本中，铜的成本占比 = 30%
- 铜价从 ¥55,000/吨 涨至 ¥66,000/吨，涨幅 = 20%
- 合理产品涨价 = 30% × 20% = **6%**

若供应商要求涨价 15%，明显超出合理范围。

**Step 3：对比市场指数**

- 铜价：上海有色金属网（SMM）、伦敦金属交易所（LME）
- 铝价：上海铝价指数
- 钢价：我的钢铁网
- 塑料粒子：中国塑料城

**Step 4：反向谈判**

- 要求供应商消化部分涨幅（供应商也有自己的成本管控能力）
- 谈判涨价有效期（如只接受半年，6个月后重新评估）
- 承诺增量订单换取供应商不涨价（以量换价）

## 六、价格构成中的税务知识 🟢 基础必掌握

中国采购报价通常有两种方式：

| 报价方式 | 含义 | 采购处理 |
|---------|------|---------|
| **含税价（Tax Included）** | 含 13% 增值税（VAT）| 多数国内采购的标准报价 |
| **不含税价（Tax Excluded）** | 不含 VAT | 部分精细核算时使用 |

**换算公式**：

$$\text{不含税价} = \frac{\text{含税价}}{1 + \text{税率}}$$

$$\text{含税价} = \text{不含税价} \times (1 + \text{税率})$$

**示例**：含税报价 ¥113，税率 13%  
→ 不含税价 = 113 ÷ 1.13 = **¥100.00**  
→ 可抵扣进项税 = 100 × 13% = **¥13.00**

> 💡 **实务要点**：比较不同供应商报价时，必须统一口径（都用含税或都用不含税），否则比较结果无效。

## 七、关键术语速查 🟢

| 术语 | 解释 |
|------|------|
| **Should Cost** | 应该成本，采购方自建的合理价格模型 |
| **DM** | Direct Material，直接材料成本 |
| **DL** | Direct Labor，直接人工成本 |
| **MOH** | Manufacturing Overhead，制造费用 |
| **Gross Margin** | 毛利率 = (售价 - 成本) ÷ 售价 × 100% |
| **PPI** | Producer Price Index，生产者价格指数，衡量出厂价格变化 |
| **LME** | London Metal Exchange，伦敦金属交易所（铜铝铅锌锡镍权威报价）|
| **应价分析** | Price Analysis，评估供应商报价合理性的系统方法 |
