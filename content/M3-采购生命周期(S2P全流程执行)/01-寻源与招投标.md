## 一、寻源（Sourcing）的触发时机 🟢 基础必掌握

不是所有采购都需要重新寻源。寻源通常在以下情况触发：

| 触发场景          | 说明                      |
| ------------- | ----------------------- |
| **全新物料/服务**   | 公司首次采购，AVL 中没有合适供应商     |
| **现有供应商无法满足** | 产能不足、质量持续不达标、价格失去竞争力    |
| **合同到期重新招标**  | 框架合同到期，重新评估市场，引入竞争      |
| **战略物料年度寻源**  | 即使合作满意，也定期开展市场询价，保持议价筹码 |
| **国产替代**      | 外资供应商受制裁或价格过高，寻找国内替代来源  |

## 二、RFI / RFQ / RFP 的区别与使用场景 🟢 基础必掌握

### 快速对照表

| 文件      | 全称                             | 用途           | 是否含报价 | 适用场景         |
| ------- | ------------------------------ | ------------ | ----- | ------------ |
| **RFI** | Request for Information（信息征询书） | 摸底市场，了解供应商能力 | ❌ 不含  | 新市场调研、新供应商初筛 |
| **RFQ** | Request for Quotation（询价书）     | 获取标准化产品报价    | ✅ 含   | 规格明确的量产物料    |
| **RFP** | Request for Proposal（征求建议书）    | 获取综合解决方案     | ✅ 含   | 复杂服务、定制化需求   |

### 流程逻辑

```
新品 / 新市场
     ↓
RFI（摸底，筛选候选供应商）
     ↓
RFQ 或 RFP（正式询价 / 征求方案）
     ↓
供应商评估与选择
     ↓
谈判 → 合同
```

## 三、RFQ 的编写要点 🟢 基础必掌握

RFQ 是采购人日常最常用的文件。一份标准 RFQ 应包含：

### RFQ 标准结构

```
1. 采购背景说明（公司简介、项目背景）
2. 物料/服务规格（技术参数、图纸、标准）
3. 数量需求（年度预计用量、本次询价数量）
4. 交货要求（交期、交货地点、包装要求）
5. 质量要求（认证要求、检验标准）
6. 报价格式要求（按指定表格填写，含单价/总价/MOQ）
7. 截止日期（供应商报价提交截止时间）
8. 保密条款（报价信息保密）
9. 联系方式
```

### 关键注意事项

- **规格描述要完整**：模糊的规格会导致供应商各自理解，报价失去可比性
- **同时发给多家**：至少 3 家，形成竞争。供应商知道有竞争，报价更认真
- **不透露竞争对手报价**：防止供应商串通或只做微小让步

## 四、RFP 的使用场景与结构 🟡 进阶了解

RFP 适用于规格不完全确定、需要供应商提出方案的采购，如：
- IT 系统采购（供应商提出架构方案）
- 工程服务（供应商设计施工方案）
- 物流服务（供应商规划配送路线和仓储方案）
- AI/SaaS 软件采购（供应商提出产品方案和定价模型）

### RFP 与 RFQ 的核心区别

| 维度    | RFQ    | RFP                  |
| ----- | ------ | -------------------- |
| 规格    | 采购方已明确 | 采购方只明确需求，规格由供应商提出    |
| 评估维度  | 主要看价格  | 价格 + 技术方案 + 实施能力综合评估 |
| 报价复杂度 | 简单（填表） | 复杂（含方案书 + 报价单）       |

---

## 五、供应商评估：技术评审（T评）与商务评审（C评）🟢 基础必掌握

收到供应商报价后，需要分两个维度评估：

### T 评（Technical Evaluation，技术评审）

**参与方**：技术/工程/质量部门
**评估内容**：
- 规格符合性（是否满足技术要求）
- 质量认证（是否有要求的认证）
- 样品/方案可行性

**结果**：Pass（通过）/ Conditional Pass（条件通过）/ Fail（不通过）

只有通过 T 评的供应商，才进入 C 评阶段。

---

### C 评（Commercial Evaluation，商务评审）

**参与方**：采购/财务部门
**评估内容**：

| 维度                 | 说明                    |
| ------------------ | --------------------- |
| **价格**             | 单价、MOQ（最小起订量）、阶梯价格    |
| **付款条件**           | 账期（Net 30/60/90）、预付比例 |
| **交货期（Lead Time）** | 正常交期、紧急交期             |
| **价格有效期**          | 报价有效期多久               |
| **年降承诺**           | 是否承诺年度价格下降            |

### 综合评分表示例

| 供应商   | T评结果 | 单价（元） | 账期     | Lead Time | 综合评分 |
| ----- | ---- | ----- | ------ | --------- | ---- |
| 供应商 A | Pass | 10.5  | Net 60 | 8周        | 85分  |
| 供应商 B | Pass | 11.0  | Net 90 | 6周        | 82分  |
| 供应商 C | Fail | —     | —      | —         | 淘汰   |

## 六、招投标法的合规要求 🟡 进阶了解

在中国，以下情况**必须**依法招投标（《招标投标法》规定）：

- 国有资金项目（政府采购、国企采购）达到一定金额门槛
- 基础设施类工程
- 特许经营项目

**民营企业自主采购**：无强制公开招标要求，但多数规范化公司内部规定：
- 超过一定金额（如 50 万元）需 3 家以上供应商竞争性报价
- 超过一定金额（如 200 万元）需高层审批

> 💡 **注意**：即使不受《招标投标法》强制约束，走规范的竞争性采购流程（多家询价、评分透明）也是职业合规的最低要求，可以保护采购人员不被审计质疑。
