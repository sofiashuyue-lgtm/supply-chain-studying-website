## 一、对账结算在 S2P 中的位置 🟢 基础必掌握

对账结算是采购闭环的最后一步，也是与财务部门协作最密切的环节。

```
采购订单 PO → 收货 GR → 供应商开具发票 → 三单匹配 → 确认应付账款 → 付款
```

**为什么采购人需要懂结算？**
- 你的 PO 是付款的依据
- PO 填错（单价/数量/税率）会导致财务无法匹配，付款卡住，供应商催款找你
- 账期管理直接影响企业现金流（DPO 指标）

---

## 二、三单匹配原则（3-Way Match）🟢 基础必掌握

付款前，财务必须核对三张单据完全一致：

```
┌──────────────┐    数量/单价    ┌──────────────┐    数量/金额    ┌──────────────┐
│  采购订单 PO  │ ←──一致──────→ │  收货单 GR   │ ←──一致──────→ │ 供应商发票   │
│ (采购部门)   │                │  (仓库)      │                │ Invoice      │
└──────────────┘                └──────────────┘                └──────────────┘
         三者一致 → 财务过账应付账款（AP）→ 到期付款
         三者不一致 → 进入异常处理流程
```

### 三单不匹配的常见原因及处理

| 不匹配类型 | 原因 | 处理方式 |
|---------|------|---------|
| **发票金额 > PO 金额** | 供应商多开、含运费未在 PO 中体现 | 与供应商确认，要求开具红字发票重开 |
| **发票数量 > GR 数量** | 供应商发票提前开，货未全到 | 等余货到厂 GR 过账后再匹配 |
| **发票税率不对** | 税率适用错误（13% vs 9%）| 要求供应商重开正确税率发票 |
| **发票信息有误** | 公司名称、税号、地址有误 | 要求供应商作废并重开 |

## 三、中国增值税发票体系 🟢 基础必掌握

中国企业的对账结算涉及增值税（VAT）发票，采购人必须基本了解。

### 发票类型

| 类型 | 含义 | 采购侧重要性 |
|------|------|------------|
| **增值税专用发票（专票）** | 可用于抵扣进项税 | ⭐⭐⭐ 最重要，必须索取 |
| **增值税普通发票（普票）** | 不可抵扣进项税 | 仅用于报销、小额采购 |
| **电子发票** | 2021年起全面推广，与纸质专票同等效力 | 逐步取代纸质 |

### 增值税抵扣逻辑（简化版）

```
企业销售商品 → 收取销项税（Output VAT）
企业采购物料 → 支付进项税（Input VAT）

实际缴纳 VAT = 销项税 - 进项税
```

**对采购的影响**：每一张进项专票都可以抵扣税款，因此：
- 必须索取增值税**专用发票**（普票无法抵扣）
- 供应商必须是**一般纳税人**（小规模纳税人开的专票抵扣额少）
- 专票必须认证（在税务系统勾选确认）才能生效

> 💡 **转行关键点**：你的工程造价背景让你对甲方/乙方结算逻辑有基础理解，增值税抵扣是同样的逻辑——只是从工程结算转移到了物料采购场景。

## 四、账期管理（DPO）🟡 进阶了解

**DPO（Days Payable Outstanding，应付账款天数）**：企业从收货到实际付款所用的平均天数。

$$DPO = \frac{应付账款余额}{采购成本/天}$$

**DPO 的战略意义**：
- DPO 越高 → 企业占用供应商资金时间越长 → 对企业现金流越有利
- DPO 越低 → 付款越快 → 供应商满意度高，但企业现金流压力大

**大厂惯用策略**：强势企业（苹果、沃尔玛）对供应商账期达 60～120 天，甚至更长，实质上是用供应商资金为自己融资。

### 中国行业 DPO 参考基准

| 行业 | 典型 DPO |
|------|---------|
| 大型零售（沃尔玛中国）| 60～90天 |
| 消费电子制造 | 45～60天 |
| 汽车主机厂 | 90～120天 |
| 中小制造企业 | 30～45天 |

## 五、早付折扣（Dynamic Discounting）🟡 进阶了解

部分企业对有现金流需求的供应商提供"早付折扣"：

**典型条款**：**2/10 Net 60** = 60天到期付款，但若在10天内付款，可享受2%折扣

**对采购方的决策逻辑**：

$$年化折扣率 = \frac{折扣率}{1 - 折扣率} \times \frac{365}{账期天数 - 折扣天数}$$

**示例**：2/10 Net 60 → 年化折扣率 ≈ 14.9%

如果企业资金成本（如贷款利率）低于14.9%，早付划算；反之不划算。

## 六、对账结算的闭环检查清单 🟢 基础必掌握

每月对账时，采购人需配合财务完成以下确认：

- [ ] 所有本月 GR 的供应商发票是否已收到？
- [ ] 是否存在"GR 已过账但发票未收"的悬挂条目？（Accrual 应计）
- [ ] 所有发票是否完成三单匹配？
- [ ] 是否有到期未付款的发票？（可能触发违约金）
- [ ] 是否有未结清的争议发票？（需跟进处理）
- [ ] 供应商是否对账期有异议？

## 七、关键术语速查 🟢

| 术语 | 全称/解释 |
|------|---------|
| **AP** | Accounts Payable，应付账款 |
| **GR** | Goods Receipt，收货/入库 |
| **IV** | Invoice Verification，发票核验 |
| **3-Way Match** | 三单匹配（PO + GR + Invoice）|
| **DPO** | Days Payable Outstanding，应付账款天数 |
| **VAT** | Value Added Tax，增值税 |
| **专票** | 增值税专用发票（可抵扣进项税）|
| **Accrual** | 应计账款，GR 已过账但发票未到的临时记账 |
| **Credit Note** | 红字发票，供应商用于冲销错误发票 |
