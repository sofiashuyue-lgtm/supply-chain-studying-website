## 一、采购合同的类型 🟢 基础必掌握

| 合同类型       | 全称                                 | 适用场景                         |
| ---------- | ---------------------------------- | ---------------------------- |
| **框架合同**   | MSA（Master Supply Agreement）       | 长期合作，约定整体商务条件，单次采购再开 PO      |
| **单次合同**   | Purchase Contract / Sales Contract | 一次性采购，合同即订单                  |
| **PO 即合同** | Purchase Order as Contract         | 供应商接受 PO 即视为接受合同条款（适用于简单标准品） |
| **保密协议**   | NDA（Non-Disclosure Agreement）      | 在正式合同前，保护双方商业信息              |

### 框架合同 vs 单次合同

| 维度 | 框架合同（MSA）| 单次合同 |
|------|-------------|---------|
| 有效期 | 通常1～3年 | 一次采购结束后失效 |
| 谈判成本 | 前期高，但之后执行简单 | 每次都需谈判 |
| 价格 | 可约定价格机制（年降条款）| 一次性锁定 |
| 适用 | 长期稳定合作的战略/杠杆供应商 | 一次性采购、样品采购 |

## 二、合同谈判的核心商务条款 🟢 基础必掌握

### 条款一：价格与调价机制

| 细节                            | 说明                       |
| ----------------------------- | ------------------------ |
| 固定价格                          | 合同期内不变，双方风险共担            |
| 浮动价格（Price Adjustment Clause） | 绑定原材料指数（如铜价、钢价），价格随指数浮动  |
| 年降条款                          | 每年按约定比例降价（参见 M1-05 谈判技巧） |
| 阶梯价格                          | 采购量越大单价越低                |

> 💡 **转行者注意**：谈"固定价格"对采购方看似有利，但如果原材料大幅下跌，供应商利润飙升，你错失了分享降价收益的机会；如原材料大涨，供应商可能违约或要求补偿。**价格机制的选择取决于你对市场走势的判断**。

---

### 条款二：付款条件（Payment Terms）🟢

| 术语                       | 含义                    |
| ------------------------ | --------------------- |
| **Net 30 / 60 / 90**     | 收货/开票后 30/60/90 天付款   |
| **预付款（Advance Payment）** | 交货前先付一部分，通常用于定制品或大额订单 |
| **货到付款（COD）**            | 收货即付，对供应商有利           |
| **见票即付（Sight Payment）**  | 收到发票后立即付款             |

**谈判原则**：
- 账期越长 → 采购方现金流越好（DPO 高），但供应商要求价格补偿
- 账期越短 → 可能换来更低价格（早付折扣）

**中国企业常规**：
- 大厂对中小供应商：Net 60～90（买方强势）
- 中小企业向大供应商：可能被要求预付 30% 或更短账期（供应商强势）

---

### 条款三：交货条款（Delivery Terms）🟢

**Incoterms 2020**（国际贸易术语）定义了买卖双方在货物运输中的责任划分（详见 M4 国际贸易模块）。

**国内采购常用**：
- **DDP（Delivered Duty Paid，完税后交货）**：供应商承担到买方仓库的一切费用和风险——对采购方最有利
- **FOB（Free On Board，装运港船上交货）**：供应商负责到指定港口上船，之后由采购方安排运输
- **EXW（Ex Works，工厂交货）**：供应商只需在自己工厂备好货，其余全由采购方负责——对供应商最省事

---

### 条款四：质量条款 🟢

| 内容                  | 说明                        |
| ------------------- | ------------------------- |
| **验收标准**            | 按哪个标准验货（企业内部标准/国家标准/行业标准） |
| **不良率上限**           | PPM 上限要求                  |
| **质量保证期（Warranty）** | 通常12～24个月，期间发生质量问题由供应商负责  |
| **来料检验（IQC）时限**     | 收货后多少天内完成检验并反馈            |
| **不合格品处理**          | 退货、换货、折价、就地销毁的权责划分        |

---

### 条款五：违约责任 🟡 进阶了解

| 违约类型          | 常见条款                              |
| ------------- | --------------------------------- |
| **供应商延误交货**   | 每日按货款金额的万分之 X 计算违约金（通常 0.5‰～3‰/天） |
| **质量不达标**     | 退货运费由供应商承担，重新发货期限，严重时赔偿停线损失       |
| **采购方延迟付款**   | 按天计算逾期违约金，给供应商对应保护                |
| **采购方单方取消订单** | NCNR（不可取消不可退）物料需补偿供应商库存损失         |

**NCNR（Non-Cancelable, Non-Returnable）**：一旦下单，采购方不得取消，供应商不接受退货。常见于：
- 定制芯片、特殊规格物料
- Lead Time 超长物料（超过16周）
- 供应商为此单专门备了原材料或产能

---

### 条款六：知识产权与保密 🟡 进阶了解

| 条款                | 内容                                                          |
| ----------------- | ----------------------------------------------------------- |
| **保密条款（NDA/NNN）** | 双方商业信息（报价、工艺、设计图）不得向第三方披露                                   |
| **NNN 协议**        | Non-Disclosure + Non-Use + Non-Circumvention，中国涉外合作的加强版保密协议 |
| **知识产权归属**        | 合作开发的技术/工艺，专利归谁所有                                           |
| **不竞争条款**         | 供应商不得将你的独家产品卖给竞争对手                                          |

## 三、合同审批流程 🟢 基础必掌握

标准合同在签署前需要经过内部审批：

```
采购起草合同草稿
      ↓
法务部门审核（合法性、风险条款）
      ↓
财务部门审核（付款条件、会计处理）
      ↓
业务/技术部门确认（规格、交期）
      ↓
授权人签署（金额越大，审批层级越高）
      ↓
双方签字盖章，合同生效
```

**审批金额分级（仅供参考，各公司不同）**：
| 金额 | 审批级别 |
|------|---------|
| < 10万元 | 采购主管 |
| 10万～100万元 | 采购总监 |
| > 100万元 | CFO / CEO |

## 四、合同签署后的管理 🟡 进阶了解

合同签完不是终点，需要持续管理：

- **合同台账**：建立合同清单，跟踪有效期、付款节点、年降到期日
- **合同变更（Change Order）**：需求变化时，通过书面变更单修订合同，不能口头修改
- **合同续签提醒**：框架合同到期前3～6个月启动续签谈判，避免断档
- **合同归档**：原件妥善保存，电子版录入系统（SRM / 合同管理系统）
