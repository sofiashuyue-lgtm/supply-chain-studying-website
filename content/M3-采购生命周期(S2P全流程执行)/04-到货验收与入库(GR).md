## 一、GR 在 S2P 中的位置 🟢 基础必掌握

**GR（Goods Receipt，收货 / 入库）**：货物到厂后，仓库和质量部门完成验收，在 ERP 系统中确认入库的操作。

GR 是采购执行阶段的关键节点：
- **触发库存增加**：GR 过账后，ERP 中的库存账正式增加
- **触发付款义务**：三单匹配（PO + GR + Invoice）中，GR 是核心环节
- **触发质量评估**：GR 过账后，物料进入 IQC 检验流程

---

## 二、到货验收的标准流程 🟢 基础必掌握

```
Step 1：预约与准备
  仓库收到 ASN → 确认到货时间 → 安排卸货区和检验资源

Step 2：外观与数量核查（仓库执行）
  ↓ 核对送货单 vs PO 编号/物料编号/数量
  ↓ 检查包装是否破损、受潮、变形
  ↓ 清点实物数量（抽查或全检）

Step 3：IQC 来料检验（质量部门执行）
  ↓ 按检验规范抽样（或全检）
  ↓ 测量、测试、功能验证

Step 4：ERP 系统过账（GR 过账）
  ↓ 录入系统：PO 号、物料编号、收货数量、质检结论
  ↓ 系统自动生成 GR 凭证（Material Document）

Step 5：物料上架入库
  货物分配货位，贴标签，完成入库
```

## 三、IQC（来料质量检验）详解 🟡 进阶了解

**IQC（Incoming Quality Control，来料质量检验）**：对供应商交付物料进行质量抽检，决定是否接收入库。

### 抽样方案：AQL 抽样标准

**AQL（Acceptable Quality Level，可接受质量水平）**：国际标准 ISO 2859，根据批量大小和质量要求，规定抽检数量和判定标准。

| AQL 等级 | 含义 | 适用场景 |
|---------|------|---------|
| AQL 0.65 | 严格，每千件允许不良 < 6.5 件 | 关键功能件、安全件 |
| AQL 1.0 | 常规严格 | 一般功能件 |
| AQL 2.5 | 一般严格 | 外观件、非关键件 |
| AQL 4.0 | 宽松 | 低风险辅材、包装 |

> 💡 **实际操作**：大多数采购人不需要自己计算 AQL，这是质量工程师的工作。但你需要知道"AQL 是什么、为什么不是每个都检、抽检比例是怎么确定的"——面试中有可能被问到。

### IQC 检验结论与处理

| 结论 | 含义 | 后续处理 |
|------|------|---------|
| **PASS（合格）** | 正常入库，ERP 过账 | 转入正常库存 |
| **WAIVED（让步接收）** | 不完全合格，但经评审可用 | 限用或返工后使用，记录在案 |
| **REWORK（返工）** | 问题可修复，送供应商或内部返工 | 返工后重新检验 |
| **REJECT（拒收）** | 不合格，退回供应商 | 发 SCAR，退货，重新发货 |

## 四、特殊收货场景处理 🟡 进阶了解

### 短交（Short Shipment）

实际到货数量少于 PO 数量：

```
部分入库（GR 实际数量）→ 通知供应商补货 → 判断是否影响生产计划
                                            ↓ 影响 → 紧急处理（见 M3-03）
                                            ↓ 不影响 → 正常等待补货
```

**注意**：ERP 中只过账实际收到的数量，PO 余量保持 Open 状态，待后续补货。

### 超交（Over Shipment）

实际到货多于 PO 数量：

- **小幅超交（< 5%）**：通常默认接收，GR 实际数量
- **大幅超交**：与供应商确认是否退回，避免产生不必要的付款义务

### 错料（Wrong Material）

收到的物料与 PO 不符（型号/规格错误）：

```
立即拒收，不进行 GR 过账
     ↓
通知供应商，要求更换正确物料
     ↓
评估对生产计划的影响，启动应急措施
```

## 五、GR 过账后的系统联动 🟢 基础必掌握

GR 完成后，ERP 系统自动触发：

```
GR 过账
  ├── 库存增加（WMS 更新库存账）
  ├── PO 状态更新（Partially/Fully Delivered）
  └── 三单匹配就绪（等待 Invoice）
        ↓
  财务应付账款（AP）可以开始匹配 Invoice
        ↓
  付款流程启动
```

## 六、关键术语速查 🟢

| 术语 | 全称/解释 |
|------|---------|
| **GR** | Goods Receipt，收货/入库 |
| **IQC** | Incoming Quality Control，来料检验 |
| **ASN** | Advanced Shipment Notification，提前发货通知 |
| **AQL** | Acceptable Quality Level，可接受质量水平 |
| **SCAR** | Supplier Corrective Action Request，供应商纠正措施请求 |
| **Waiver** | 让步接收，不合格但经审批可用 |
| **GRN** | Goods Receipt Note，收货单据（与 GR 同义）|
| **FIFO** | First In First Out，先进先出，库存管理原则 |
