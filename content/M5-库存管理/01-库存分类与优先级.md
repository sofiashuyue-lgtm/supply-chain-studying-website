# 01 库存分类与优先级

> **📌 知识层级说明**
> 🟢 **基础必掌握** — 入职前、面试前必须理解
> 🟡 **进阶了解** — 有基础后扩展，理解逻辑即可
> 🔴 **暂缓** — 工作后遇到实际场景再深入

## 一、为什么要对库存分类？🟢 基础必掌握

一个中型制造企业可能管理 **5,000～50,000 个 SKU**（库存单品）。对每个 SKU 投入同样的管理精力，既不可能也不高效。

**库存分类的目的**：
- 识别最重要的物料（高价值、高风险），给予优先管理
- 对低价值物料降低管理成本（批量采购、自动补货）
- 减少呆滞库存，优化现金流

> 💡 **转行迁移**：工程造价中的重点工程量清单分类（主体结构 vs 装饰装修 vs 水电安装）与此逻辑相同——优先精确核算高价值项目，低价项目用估算即可。

## 二、ABC 分类法（帕累托原则）🟢 基础必掌握

**ABC 分类法**基于帕累托原则（Pareto Principle，二八定律）：

```
A 类（重要）：约 20% 的 SKU，占总采购金额的 70%～80%
B 类（次要）：约 30% 的 SKU，占总采购金额的 15%～20%
C 类（一般）：约 50% 的 SKU，占总采购金额的 5%～10%
```

### ABC 分类的实施步骤

```
Step 1：计算每个 SKU 的年度采购金额
         = 年用量 × 单价

Step 2：按金额从高到低排序

Step 3：计算累计金额占比

Step 4：划定 ABC 区间
         累计占比 0～70%   → A 类
         累计占比 70%～90% → B 类
         累计占比 90%～100% → C 类
```

### 各类物料的管理策略

| 类别 | 管理策略 | 补货方式 | 安全库存 | 盘点频率 |
|------|---------|---------|---------|---------|
| **A 类** | 精细管理，高度关注 | 按需订购，小批量高频次 | 严格计算，精确控制 | 每月或每周 |
| **B 类** | 定期关注，适度库存 | 定期订购（EOQ 模型）| 标准计算 | 每季度 |
| **C 类** | 简化管理，批量订购 | 大批量低频次，设定固定补货点 | 宽松设定，防止缺货即可 | 每半年 |

## 三、ABC-XYZ 矩阵分析 🟡 进阶了解

**XYZ 分类**在 ABC 基础上叠加**需求稳定性**维度：

| X 类 | 需求非常稳定，可预测性高（变异系数 < 20%）|
|------|----------------------------------------|
| **Y 类** | 需求有一定波动，可预测性中等（变异系数 20%～50%）|
| **Z 类** | 需求高度波动，难以预测（变异系数 > 50%）|

**变异系数（CV，Coefficient of Variation）**：
$$CV = \frac{\sigma}{\mu} \times 100\%$$

其中 σ 为需求标准差，μ 为平均需求。

### 9 格矩阵与管理策略

|  | X（稳定）| Y（波动）| Z（不稳定）|
|--|---------|---------|-----------|
| **A（高价值）** | AX — 精细管理，MRP 计划驱动，安全库存低 | AY — 加强预测，保持适度安全库存 | AZ — 高度关注，保持较高安全库存，考虑多供应商 |
| **B（中价值）** | BX — 标准管理，定期补货 | BY — 定期回顾库存策略 | BZ — 按需采购为主 |
| **C（低价值）** | CX — 简化管理，大批量订购 | CY — 减少库存，提高订货频次 | CZ — 降低持有，考虑JIT或Consignment（寄售）|

### 实务应用重点

**AZ 类**是管理难点：价值高（不能缺货）但需求不稳定（难以预测）
→ 对策：
1. 保持较高安全库存（接受较高持库成本）
2. 开发第二货源（减少供应风险）
3. 与销售团队建立需求共享机制（更早获取需求信号）

**CX 类**是管理重点之外的对象：
→ 对策：大批量采购 + 自动补货（kanban看板补货），几乎不用人工介入

## 四、其他库存分类维度 🟡 进阶了解

### 按库存功能分类

| 类型 | 说明 | 采购场景 |
|------|------|---------|
| **循环库存（Cycle Stock）** | 正常周转的库存，在补货周期内消耗 | 常规采购订单的在库存量 |
| **安全库存（Safety Stock）** | 应对需求波动和供应不确定的缓冲库存 | 详见 M5-02 |
| **在途库存（In-Transit）** | 已下单但尚未到货的库存 | PO 发出后、GR 前的状态 |
| **寄售库存（Consignment）** | 供应商库存存放在买方仓库，用多少付多少 | VMI/Hub 模式下的重要库存形式 |
| **呆滞库存（Slow-Moving/Dead Stock）** | 长期无消耗的库存 | 详见 M5-04 |

### 按物料属性分类（制造业）

| 类型 | 说明 |
|------|------|
| **原材料（Raw Material）** | 未加工的基础物料 |
| **在制品（WIP，Work In Process）** | 正在生产过程中的半成品 |
| **产成品（Finished Goods）** | 可直接销售的成品 |
| **MRO（Maintenance, Repair, Operations）** | 维修、保养、运营消耗品，非直接生产物料 |

## 五、关键术语速查 🟢

| 术语 | 解释 |
|------|------|
| **SKU** | Stock Keeping Unit，库存单品，每个独立的物料品种 |
| **ABC 分类** | 按价值从高到低分三类，集中精力管理高价值物料 |
| **XYZ 分类** | 按需求稳定性分三类，稳定/波动/不稳定 |
| **CV** | Coefficient of Variation，变异系数，衡量需求波动程度 |
| **Pareto 原则** | 二八定律，20% 的 SKU 贡献 80% 的价值 |
| **Consignment** | 寄售，供应商货存买方仓，使用时才付款 |
| **MRO** | 间接物料，维修保养运营消耗品 |
| **Kanban** | 看板，一种拉式补货触发机制 |
