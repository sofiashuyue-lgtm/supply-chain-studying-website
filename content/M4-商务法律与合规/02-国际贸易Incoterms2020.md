## 一、为什么采购人要懂 Incoterms？🟢 基础必掌握

**Incoterms（International Commercial Terms，国际贸易术语）** 是由国际商会（ICC）制定的标准化贸易术语体系，最新版本为 **Incoterms 2020**（2020年1月1日生效）。

它解决的核心问题：**谁负责运费？谁承担风险？风险在哪个节点转移？**

在采购工作中：
- 与境外供应商签合同，必须明确 Incoterms 条款
- 不同条款直接影响采购成本（到港价 vs 到厂价差异可达 5%～15%）
- 影响进口清关责任方、保险责任方
- 影响 Landed Cost（落地成本）的计算

> 💡 **转行迁移**：工程造价中的"甲供材"（甲方供货到场）≈ DDP；"乙方自采"（乙方自行采购运到）≈ EXW 或 FCA。逻辑完全一样：谁管运输、谁担风险。

## 二、Incoterms 2020 全览：11个条款 🟢 基础必掌握

### 按适用运输方式分类

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【任何运输方式均适用】（7个）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
EXW  → FCA → CPT → CIP → DAP → DPU → DDP
（买方责任越来越小 →）      （卖方责任越来越大 →）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【仅适用海运/内河水运】（4个）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
FAS  →  FOB  →  CFR  →  CIF
```

### 风险转移节点一览

| 条款 | 全称 | 风险转移节点 | 运费谁付 | 保险谁买 |
|------|------|------------|---------|---------|
| **EXW** | Ex Works（工厂交货）| 卖方工厂门口 | 买方 | 买方 |
| **FCA** | Free Carrier（货交承运人）| 约定地点交给承运人 | 买方 | 买方 |
| **CPT** | Carriage Paid To（运费付至）| 交给承运人时 | 卖方（到目的地）| 买方 |
| **CIP** | Carriage and Insurance Paid To | 交给承运人时 | 卖方 | 卖方（须保 ICC-A 最高险）|
| **DAP** | Delivered at Place（目的地交货）| 目的地卸货前 | 卖方 | 买方 |
| **DPU** | Delivered at Place Unloaded（卸货后交货）| 目的地卸货后 | 卖方 | 买方 |
| **DDP** | Delivered Duty Paid（完税后交货）| 目的地指定地点 | 卖方（含关税）| 卖方 |
| **FAS** | Free Alongside Ship（船边交货）| 装运港船边 | 买方 | 买方 |
| **FOB** | Free On Board（船上交货）| 装运港货物越过船舷 | 买方 | 买方 |
| **CFR** | Cost and Freight（成本加运费）| 装运港货物越过船舷 | 卖方（运费）| 买方 |
| **CIF** | Cost, Insurance and Freight | 装运港货物越过船舷 | 卖方（运费）| 卖方（最低保险）|

## 三、采购最常用的 4 个条款详解 🟢 基础必掌握

### EXW — 工厂交货（对买方最不利）

```
卖方工厂 ──[风险转移]──→ 买方安排提货 → 出口清关 → 国际运输 → 进口清关 → 到厂
                          ↑
                   买方承担一切责任和费用
```

**适用场景**：买方有强大的物流能力，想控制整个运输过程以降低成本。  
**陷阱**：卖方甚至不负责出口清关！实务中国内贸易用 EXW 常见，跨境慎用。

---

### FOB — 船上交货（最经典的出口条款）

```
卖方工厂 → 出口清关（卖方）→ 货物上船 ──[风险转移]──→ 国际运输（买方）→ 进口清关（买方）→ 到厂
```

**适用场景**：买方自行安排国际货运，可以凭规模优势拿到更低运价。  
**注意**：FOB 仅适用于海运，不适用于空运和多式联运（应改用 FCA）。

**价格构成**：
```
FOB 价 = 工厂出货价（EXW）+ 内陆运输费 + 出口清关费 + 港口装船费
```

---

### CIF — 成本+保险+运费（最常见的进口报价方式）

```
卖方工厂 → 出口清关（卖方）→ 货物上船 ──[风险转移]──→ 卖方付运费+保险 → 到港
                                                                           买方负责进口清关+到厂运输
```

**适用场景**：买方不想处理国际运输细节，让卖方报到港价格。  
**注意**：CIF 下保险由卖方购买，但**实际受益人是买方**。注意保险条款是否充分（CIF 默认仅最低保险 ICC-C，建议要求 ICC-A 全险）。

**价格构成**：
```
CIF 价 = FOB 价 + 国际运费 + 保险费
到岸价（Landed Cost）= CIF 价 + 进口关税 + 增值税 + 进口代理费 + 内陆运费
```

---

### DDP — 完税后交货（对买方最有利）

```
卖方工厂 → 出口清关（卖方）→ 国际运输（卖方）→ 进口清关（卖方，含关税）→ 到指定目的地 ──[风险转移]──→ 买方
```

**适用场景**：买方完全不想处理进口环节，让卖方报门到门价格（Door to Door）。  
**注意**：对卖方责任最重；买方需验证卖方是否有进口资质（部分国家外国企业不能直接进口）。

## 四、Incoterms 2020 的新变化 🟡 进阶了解

相比 Incoterms 2010，2020 版主要变化：

| 变化点 | 2010 版 | 2020 版 |
|--------|---------|---------|
| **DAT → DPU** | DAT（目的地码头交货）| 改名 DPU（目的地卸货后交货），适用地点扩展至任何地点 |
| **CIP 保险要求提高** | 最低保险（ICC-C）| 要求 ICC-A（最高级全险），更保护买方 |
| **FCA 与信用证** | 实务操作不便 | 允许买方要求船方签发已装船提单，解决信用证付款问题 |
| **自有运输方式** | 隐含在 FCA/DAP | 明确允许使用自有运输工具（如企业自营物流）|

## 五、采购报价比较：不同条款的成本计算 🟡 进阶了解

**场景**：从中国深圳采购一批电子元件运到美国洛杉矶，两家供应商报价：
- 供应商A：FOB 深圳港，USD 10,000
- 供应商B：CIF 洛杉矶港，USD 11,200

**计算 Landed Cost 对比**：

| 费用项目 | 供应商A（FOB）| 供应商B（CIF）|
|---------|------------|------------|
| 货物价格 | 10,000 | — |
| 国际海运费 | +800（买方安排）| 含在内 |
| 保险费 | +100（买方安排）| 含在内 |
| CIF 小计 | 10,900 | 11,200 |
| 进口关税（假设5%）| +545 | +560 |
| 进口增值税 | — | — |
| 内陆运费（LA港→仓库）| +300 | +300 |
| **到厂总成本** | **11,745** | **12,060** |

→ **供应商A（FOB）实际更便宜**，尽管报价数字更低，买方自己安排运输仍划算。

## 六、关键术语速查 🟢

| 术语 | 解释 |
|------|------|
| **Incoterms** | International Commercial Terms，国际贸易术语（ICC制定）|
| **风险转移** | Risk Transfer，货物损坏/丢失的责任从卖方转向买方的时间点 |
| **Landed Cost** | 落地成本，货物到达目的地的全部总成本（含税、运、保）|
| **ICC** | Institute Cargo Clauses，货物运输保险条款；A=全险，B=中险，C=基本险 |
| **提单** | Bill of Lading（B/L），海运货物所有权证明文件 |
| **进口清关** | Import Customs Clearance，进口报关，完成才能提货 |
| **关税** | Customs Duty，进口商品向海关缴纳的税款 |
| **FOB 价** | Free on Board，货物装上船时的离岸价格 |
