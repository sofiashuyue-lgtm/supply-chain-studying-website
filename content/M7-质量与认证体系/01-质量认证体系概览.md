## 一、质量认证对采购的意义 🟢 基础必掌握

质量认证是供应商准入（AVL）的**基础门票**。没有认证，意味着：
- 质量管理体系没有经过第三方验证
- 无法证明产品符合行业规范
- 部分行业（汽车、医疗）法规要求**必须**有认证才能供货

**采购人需要了解认证的原因**：
- 审核供应商文件时，需要验证认证真实性（证书是否在效期内）
- 理解认证等级对应什么质量能力
- 面试被问"你们如何保证来料质量"时，认证体系是重要答案

## 二、ISO 9001 — 通用质量管理体系 🟢 基础必掌握

**ISO 9001（Quality Management System，质量管理体系）**：国际标准化组织（ISO）制定的通用质量管理标准，适用于所有行业，最新版本为 **ISO 9001:2015**。

### 核心理念：PDCA + 风险思维

```
Plan（策划）→ Do（执行）→ Check（检查）→ Act（改进）
```

ISO 9001:2015 在传统 PDCA 基础上，新增了**基于风险的思维（Risk-Based Thinking）**：要求企业识别影响质量的内外部风险，并采取预防措施。

### ISO 9001 的七大质量管理原则

| 原则 | 简要说明 |
|------|---------|
| **以顾客为关注焦点** | 满足并超越顾客要求 |
| **领导作用** | 最高管理层亲自主导质量文化 |
| **全员参与** | 每个人都对质量负责 |
| **过程方法** | 按流程管理，而非孤立管理各环节 |
| **改进** | 持续改进是永恒主题 |
| **循证决策** | 基于数据和证据做决策 |
| **关系管理** | 管理好与供应商等相关方的关系 |

### 适用范围与认证价值

- 进入大型制造企业 AVL 的**基础要求**（几乎所有行业）
- 获证后每年接受外部机构（如 SGS、TÜV、BV）审核维持
- **不代表产品合格**，只代表质量管理过程规范

## 三、IATF 16949 — 汽车行业质量体系 🟡 进阶了解

**IATF 16949（International Automotive Task Force，国际汽车工作组）**：汽车行业专用质量管理体系标准，在 ISO 9001 基础上叠加汽车行业特殊要求。

**适用范围**：
- 为整车厂（OEM）供应零部件的 Tier 1（一级供应商）
- Tier 1 要求其 Tier 2（二级供应商）也获得认证

### IATF 16949 vs ISO 9001 的主要区别

| 对比维度 | ISO 9001 | IATF 16949 |
|---------|---------|-----------|
| **适用行业** | 所有行业 | 汽车行业 |
| **特殊工具要求** | 无 | 强制要求 APQP、PPAP、MSA、SPC、FMEA |
| **客户特殊要求（CSR）** | 无 | 必须满足大众、宝马、通用等整车厂的特定要求 |
| **变更管理** | 一般要求 | 极严格（零件或工艺变更须重新完成 PPAP）|
| **PPM 目标** | 无明确数值 | 通常 < 50 PPM |

### 汽车行业五大工具（Core Tools）

这5个工具是 IATF 16949 的精髓，采购人至少要了解名称和用途：

| 工具 | 全称 | 用途 |
|------|------|------|
| **APQP** | Advanced Product Quality Planning | 先期产品质量策划，新品开发阶段的质量规划 |
| **PPAP** | Production Part Approval Process | 生产件批准程序，供应商证明量产能力的文件包 |
| **MSA** | Measurement System Analysis | 测量系统分析，验证检测设备的准确性 |
| **SPC** | Statistical Process Control | 统计过程控制，用控制图监控生产过程稳定性 |
| **FMEA** | Failure Mode and Effects Analysis | 失效模式与影响分析，提前识别潜在失效风险 |

> 🔴 **暂缓**：IATF 16949 五大工具的详细内容，进入汽车行业后再深入学习。

## 五、RoHS / REACH — 环保合规认证 🟢 基础必掌握

电子和制造业采购必须了解的两个环保法规：

### RoHS（有害物质限制指令）

**RoHS（Restriction of Hazardous Substances）**：欧盟指令，限制电子电气设备中6种有害物质：

| 受限物质 | 最大允许含量 |
|---------|------------|
| 铅（Pb）| 0.1% |
| 汞（Hg）| 0.1% |
| 镉（Cd）| 0.01% |
| 六价铬（Cr VI）| 0.1% |
| 多溴联苯（PBB）| 0.1% |
| 多溴二苯醚（PBDE）| 0.1% |

**对采购的影响**：
- 采购物料须要求供应商提供 **RoHS 符合性声明**
- 如有 ROHS 豁免需求，须申请豁免并存档

### REACH（化学品注册评估授权限制）

**REACH（Registration, Evaluation, Authorisation and Restriction of Chemicals）**：欧盟化学品法规，范围比 RoHS 更广，管控 200+ 种高关注物质（SVHC）。

- 采购方须向供应商要求提供 **REACH SVHC 符合性声明**
- 如产品含 SVHC 且含量 > 0.1%，须向客户披露

> 💡 **出口欧盟的中国制造企业**：这两项认证是硬性要求，供应商准入时须核查。

## 六、认证证书的验证方法 🟢 基础必掌握

收到供应商提交的认证证书，如何验证真实性？

| 步骤 | 操作 |
|------|------|
| **查证书有效期** | 证书上有明确的到期日，注意是否已过期 |
| **核实认证范围** | 证书的认证范围是否覆盖供应商为你生产的产品/工厂地址 |
| **在线验证** | 访问颁证机构（如 TÜV、SGS、BV）官网查询证书编号 |
| **核查颁证机构资质** | 颁证机构是否为 IAF（国际认可论坛）成员，是否具备合法资质 |

> ⚠️ **注意**：中国存在少量伪造认证证书的案例，尤其是中小供应商。重要供应商的认证须通过官方网站核实。

## 七、关键术语速查 🟢

| 术语 | 解释 |
|------|------|
| **ISO 9001** | 国际通用质量管理体系标准（2015版）|
| **IATF 16949** | 汽车行业质量管理体系，ISO 9001 的汽车版 |
| **PPAP** | Production Part Approval Process，生产件批准程序 |
| **APQP** | Advanced Product Quality Planning，先期产品质量策划 |
| **RoHS** | 有害物质限制指令（欧盟电子产品环保要求）|
| **REACH** | 化学品注册评估法规（欧盟化学品管控）|
| **SVHC** | Substances of Very High Concern，高关注物质 |
| **TÜV** | 德国莱茵检测机构（全球权威第三方认证机构）|
| **CSR** | Customer Specific Requirements，整车厂客户特定要求 |
